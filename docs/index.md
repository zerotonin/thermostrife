# ThermoStrife Documentation

**Historical-uprisings climate companion to
[ThermoKourt](https://github.com/zerotonin/thermokourt).** Tests whether
positive temperature anomalies are over-represented on days of mass civil
strife, using a case-crossover null built from matched non-event days at
the same station and calendar month.

## Overview

ThermoStrife curates a 112-event panel of violent uprisings, revolutions,
massacres, and prison revolts (1750–2026) and tests the heat-aggression
hypothesis on the human-data side of the ThermoKourt programme. See the
{doc}`methods` page for the pre-registered analysis plan and the
{doc}`pipeline` guide for end-to-end usage.
